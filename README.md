🏏 IPL Data Analysis Dashboard (Excel)
📌 Project Overview

This project is an interactive IPL Data Analysis Dashboard built using Microsoft Excel. It provides meaningful insights into IPL matches, team performance, and player statistics through dynamic visualizations and KPIs.

🎯 Objectives
Analyze IPL match data effectively
Track team and player performance
Identify trends and patterns in matches
Present insights using an interactive dashboard
📊 Features
📈 Team Performance Analysis
🏆 Player Statistics (Runs, Wickets, etc.)
📅 Match-wise Insights
📌 Key Performance Indicators (KPIs)
🔍 Interactive Filters & Slicers
🛠️ Tools & Technologies Used
Microsoft Excel
Pivot Tables
Charts & Graphs
Slicers & Filters
Data Cleaning & Processing
📂 Dataset

The dataset contains IPL match details including:

Teams
Players
Match results
Runs & wickets
